## Installing Nilla
Install the Nilla configuration files to your setup using the following code:
```sh
$ git clone https://github.com/transicle/Nilla
$ cd ./Nilla
$ chmod +x ./Install-Nilla.sh
$ ./Install-Nilla.sh
```
